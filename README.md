# 🏋️‍♂️ 생활체육지도사 2급(보디빌딩) 실기 암기노트 SPA

> **2026년도 국가체육지도자(생활·노인·유소년) 자격검정 실기시험 완벽 대비를 위한 모바일 최적화 웹앱**
>
> 헬스장 및 시험장 현장에서 한 손으로 심사위원 평가 문항을 체크하고, 헷갈리는 자세를 마킹하여 PC/모바일에서 연동 복습할 수 있도록 구현된 바닐라 JS 기반의 단일 페이지 애플리케이션(SPA)입니다.

## ✨ Key Features (핵심 기능)

- **모바일 퍼스트 UX/UI:** 헬스장 및 이동 중에 한 손으로 조작하기 편리하도록 터치 영역 확장 및 인터랙션 최적화.
- **분할 루틴 탭 필터링:** 시험 빈출 조합인 `[등 + 이두]`, `[가슴 + 삼두]` 및 부위별(어깨, 복부, 하체) 실전 기술 매핑.
- **📌 오답노트 (헷갈리는 자세 핀 기능):** 현장에서 정확한 자세나 호흡법이 헷갈리는 종목을 토글 배지 형태로 마킹 가능.
- **LocalStorage 영속성 레이어:** 브라우저 로컬 스토리지를 활용하여 별도의 백엔드 데이터베이스 없이도 체크 상태 및 북마크 데이터 보존.
- **실시간 진척도 게이지:** 전체 심사 항목 대비 현재 암기 완료율을 상단 프로그레스 바를 통해 시각화.
